# Matriz inicial de questões — Arena do Conhecimento — 6º ano

## Escopo da versão 1

A base inicial contém 300 questões objetivas, cada uma com quatro alternativas, gabarito, explicação pedagógica, habilidade, tema, dificuldade e cenário da Arena.

| Componente curricular | Questões |
|---|---:|
| Língua Portuguesa | 50 |
| Matemática | 50 |
| Ciências | 35 |
| História | 30 |
| Geografia | 30 |
| Língua Inglesa | 25 |
| Língua Espanhola | 25 |
| Arte | 20 |
| Educação Física | 20 |
| Ensino Religioso | 15 |
| **Total** | **300** |

## Critérios curriculares

A estrutura foi organizada a partir da Base Nacional Comum Curricular e do Currículo de Referência Único do Estado do Acre. A Língua Espanhola foi identificada com códigos curriculares locais `AC06LE...`, pois não integra a organização nacional obrigatória da BNCC da mesma forma que a Língua Inglesa.

## Validação pedagógica necessária

A base é uma primeira versão para implantação técnica. Antes de definir a distribuição por bimestre, a coordenação e os professores devem confrontar os temas com:

1. o plano de curso adotado pela escola;
2. a sequência didática do bimestre;
3. os conteúdos já ensinados em cada turma;
4. a linguagem e o nível real dos estudantes;
5. eventuais particularidades do Currículo do Acre e do projeto pedagógico da escola.

O arquivo CSV foi incluído para facilitar essa revisão no Excel. O JSON é o arquivo usado pelo importador.

## Distribuição nos cenários

- Biblioteca: Língua Portuguesa, Língua Inglesa e Língua Espanhola.
- Laboratório: Matemática e Ciências.
- Zoológico: Ciências.
- Prefeitura: História e Geografia.
- Museu: História e Arte.
- Floresta: Ciências e Geografia.
- Escola Militar: Educação Física e Ensino Religioso.
- Praça Central: questões mistas para completar a meta diária.
